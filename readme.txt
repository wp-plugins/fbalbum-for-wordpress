== Description ==

Sharing your facebook photo albums in your wordpress has never been this easier. you can access the list of you photo album right from you post/page editor. you can select any one from the list and this plugin will do rest of the tasks for you. 

Major new features in FBAlbum for WordPress 1.0 include:

* you can share any of your facebook photo album on your wordpress site.
* plugin generates shortcode for each of the albums for use in wordpress post/page.

PS: You'll need an Facebook API keys to use it.  

== Installation ==

Upload the FBAlbum for WordPress plugin to your blog, Activate it, then enter your APP ID and APP Secret in the settings page. you can collect them
from https://developers.facebook.com/apps. on this page, click on "Create New App" button and fill out the required information. 

At the same time, copy the "reditect url" in the plugin settings page and enter in the  "site URL" field in https://developers.facebook.com/apps.

1, 2, 3: You're done!
